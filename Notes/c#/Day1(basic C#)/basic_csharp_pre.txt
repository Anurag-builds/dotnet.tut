types of variable
1.value type
2.reference type

value type:
1.data store in stack memory.
eg.int


reference type:
1.data store in heap memory,address of that heap memory is stored in stack.
eg.string,object,class,delegate,array

local variable
instance variable/member/object variable

local variable:declare inside method,so scope of the local variable is within the method only.
instance variable:declare inside the class,outside of all methods,
instance variables are accessible to all methods within that class.
