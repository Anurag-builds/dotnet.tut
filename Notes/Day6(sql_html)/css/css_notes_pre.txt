types of selectors
1.universal  *
2.type
3.id  #
4.class  .
5.grouping  ,
6.combination  space

1.universal selector
apply to all elements,it is given by *
eg. 
	*
        {
           font-family:Arial;
        }

2.type selector
apply to specific type
eg.
	p
        {
           font-family:Arial;
        }

3.id selector
apply to single element,it is given # symbol
eg.
 	#demo
        {
           background-color:yellow;
           color:blue;
        }

<h1 id="demo">CSS</h1>

4.class selector
apply to different types of elements,it is given by . symbol
eg.
 	.demo
        {
           background-color:yellow;
           color:blue;
        }
<h1 class="demo">CSS</h1>
<p class="demo">welcome to CSS</p>

5)grouping
apply to different types of elements,it is given by ,
eg.
